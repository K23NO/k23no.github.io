https://github.com/K23NO/interfaz_movie_p2
Clustering de peliculas - FALL 2025-2 - Machine learning. En este proyecto se implemento diversad tecnias de clusting para agrupar peliculas segun sus caracteristicas visuales, previamente obtenidas mediante tecnicas de vision por computadora. Se utilizaron algoritmos como K-means, DBSCAN. Del cual el mejor fue el K-means. 
La interfaz desarollado en streamlit permite al usuario explorar las agrupaciones de peliculas y obtener recomendaciones basadas en sus preferencias.



https://github.com/K23NO/IntroSisEmbebidos.git
Sistema embebido - Espirometro BPM- 2025-1 - Sistemas embebidos. En este proyecto se desarollo un sistema embebido mediante la comunicación entre un microcontrolador Atmega328P y PSoC 5LP (ARM Cortex-M3), para poder medir mediante un sensor de flujo la capacidad pulmonar de un paciente. Además 
para medir la presion arterial mediante un sensor MAX30102 . El sistema incluye sensores de flujo de aire y presión, y una interfaz de usuario para mostrar los resultados en tiempo real.

Comunicación entre microcontroladores utilizando protocolo I2C y UART.
